# A numerical study of the "Output Sensitivity Modificaiton (OSM)"-strategy for dynamic optimizaiton

The OSM strategy allows dynamic optimizaiton with preferred input-output sensitivity structure. This study accopanies a paper which proposes this method, and provides detail of:
- Implementation
- Real-time applicability
-   solve times
-   iteration count
-   warm-startability
- Behavior compared to cost-function tuning methods

It also provides means for readers to do their own experiments, comparisons and exloration of the method.
